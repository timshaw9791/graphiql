服务器端需要的配置
=========

目录下应该有
    vendor文件夹
    graphiql.css
    graphiql.js
    index.html
    以及这个readme

第一次下载：
    1.将server_res下的所有文件 复制到 后端项目中的resource下

    2.重启自己的服务器

第二次更新：

    如果涉及到CSS样式的变更
    下载server_res下的graphiql.js和graphiql.css文件（CSS文件较小，可以直接复制粘贴里面的内容），
    没有则
    直接下载server_res下的graphiql.js文件，
